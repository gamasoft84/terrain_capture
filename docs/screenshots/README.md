# Capturas para el README

Los archivos `01-*.svg` … `04-*.svg` son **marcadores de posición** para que el [`README.md`](../README.md) muestre una galería sin rutas rotas.

Para documentación real:

1. Toma capturas en el dispositivo o escritorio (PNG o WebP).
2. Sustituye cada SVG por un archivo con el **mismo nombre base** y extensión `.png` (u homónimo `.webp`) y actualiza las rutas en el README si cambias el formato.

Sugerencia de set mínimo: dashboard, captura en campo, mapa del terreno, pantalla de reporte PDF/export.
